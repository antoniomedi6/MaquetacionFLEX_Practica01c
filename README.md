# MaquetacionFLEX_Practica01c

## Descripción del proyecto
Es una web para clase usando **Flexbox**. Va de una galería de fotos de caballos en el Campillo (Almería). 

## Contenido de la publicación
Es la **Práctica 04** de la asignatura de Diseño de Interfaces (DIW) de 2º de DAW.

## Desarrollo del proyecto
* He usado etiquetas de HTML5 como `section` y `figure`.
* He maquetado todo con **Flexbox** para que se mueva bien.
* He puesto iconos de redes sociales con FontAwesome.
* Los colores son verdes y azules para que peguen con el tema.
